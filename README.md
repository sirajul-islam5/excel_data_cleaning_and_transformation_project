**🧹 Data Cleaning & Transformation Project** 

Transformed complex, unformatted raw datasets into structured, accurate, and analysis-ready formats — ensuring data integrity and supporting reliable reporting/MIS processes.

**🎯 Objective:**
Real-world data is messy. This project demonstrates the full data cleaning pipeline in Microsoft Excel — from identifying quality issues in raw data to delivering a structured, professional dataset ready for analysis and reporting.

## 🔍 Before vs After:
- ✅ Date errors (######) → Corrected date formats
- ✅ ALL CAPS text → Standardized using PROPER(), UPPER(), LOWER()
- ✅ #DIV/0! errors → Identified and resolved
- ✅ Combined Department & Region → Separated using Text to Columns & Flash Fill
- ✅ Missing/NA values → Handled and documented
- ✅ Duplicate records → Identified and removed
- ✅ Inconsistent client names → Normalized for professional reporting
- ✅ Unformatted currency → Cleaned and standardized

**🛠️ Tools & Techniques Used:**
- Microsoft Excel — Text Functions, Flash Fill, Text to Columns
- Power Query — ETL (Extract, Transform, Load) process
- Conditional Formatting — Highlighting errors and inconsistencies
- Data Validation — Ensuring input accuracy and consistency
- Error Handling — #DIV/0!, NA, ###### fixes

**💡 Key Skills Demonstrated:**
- Data Transformation — Splitting combined columns for better granularity
- Data Standardization — Uniform formatting across all fields
- Error Detection & Resolution — 100% data accuracy achieved
- ETL Process — Raw data → Clean, analysis-ready output
- MIS Readiness — Cleaned dataset structured for direct use in reporting
